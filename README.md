# 💸 Expense Tracker MCP Server

A **Model Context Protocol (MCP) server** for managing personal expenses using **Python + SQLite**.  
It connects with an MCP client and provides complete expense management features.

---

## 🚀 Features

- ➕ Add Expense  
- ✏️ Edit Expense  
- ❌ Delete Expense  
- 📖 Read All Expenses  
- 📊 Category-wise Summary  
- 🗄️ SQLite Database Integration  

---

## 🧠 Project Overview

This project is an **MCP server backend** for an expense tracking system.

It allows:
- Managing expenses through MCP tools  
- Storing data in a lightweight **SQLite database**  
- Easy integration with **AI agents / MCP clients**

---

## 🏗️ Tech Stack

- **Python**
- **SQLite**
- **MCP (Model Context Protocol)**
- **FastMCP**

## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/moteprem4-web/expense-tracker-mcp.git

# Go to project folder
cd expense-tracker-mcp

# Create virtual environment
python -m venv venv

# Activate environment
venv\Scripts\activate      # Windows
source venv/bin/activate   # Mac/Linux

# Install dependencies
pip install -r requirements.txt
